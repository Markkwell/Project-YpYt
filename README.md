# Your path Your trials

Система анализа отзывов игроков Steam и распределение их на положительныве и негативные

# Проект позволяет:
-анализировать интересы игроков

-определять, чего луше избегать в играх

-находить популярные и актуальные темы
# Актуальность:
проект актуален особенно сейчас, потому, что нынешние технологии позволяют разнообразить игровой процесс и сделать его индивидуальным для каждого, помимо того, сейчас стоит острый вопрос "не того результата", которого ожидает игрок
# Возможности:
- Анализ позитивных и негативных отзывов
  
- Очистка текста отзывов
  
- Анализ слов-триггеров

- - Классификация отзывов с помощью ML
# Используемые датасеты:
https://www.kaggle.com/datasets/andrewmvd/steam-reviews/data

# Технологии

Проект написан на Python
Использованны библиотек:
- pandas

- numpy

- matplotlib

- scikit-learn
